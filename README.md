# RC Car Mechatronics Project

## Overview
This project is a mechatronics-based RC car project focused on the design, testing and optimization of a small remote-controlled vehicle.

The goal of the project was to apply basic mechatronics concepts by combining mechanical design, electronics, motor control and system behavior analysis.

## Main Features
- RC car build and system assembly
- Motor control and basic vehicle movement
- Testing of driving behavior
- Basic analysis of vehicle response
- Mechanical and electrical system integration
- Practical application of mechatronics principles

## Technologies and Tools Used
- Mechatronics system design
- DC motor control
- Basic electronics
- MATLAB / Simulink
- Solid Edge / SolidWorks
- Mechanical design
- System testing and optimization
